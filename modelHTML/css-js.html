1. Яка роль HTML, CSS та JavaScript у створенні веб-сторінки?

HTML (HyperText Markup Language) відповідає за структуру сторінки. CSS (Cascading Style Sheets) відповідає за стиль і вигляд сторінки. JavaScript використовується для реалізації динамічності і взаємодії на сторінці.

2. Що робить атрибут `style` у HTML?

Атрибут `style` в HTML використовується для визначення стилів елемента без використання зовнішнього CSS. Наприклад: <p style="color: red;">Це червоний текст</p>.

3. Як можна підключити зовнішній та внутрішній файл CSS до вашого HTML-документа?

Для зовнішнього CSS використовуйте тег `<link>`: <link rel="stylesheet" href="styles.css">. Для внутрішнього CSS використовуйте тег `<style>` в `<head>` секції або атрибут `style` в тегах елементів.

4. Що робить функція `addEventListener` у JavaScript?

Функція `addEventListener` в JavaScript використовується для додавання обробників подій до елементів. Наприклад, element.addEventListener('click', function(){ /* ваш код */ }); додасть обробник події "клік" до елемента.

Практичне питання: Створіть HTML-документ з кнопкою та сповіщенням

Створіть HTML-документ з кнопкою. При натисканні на кнопку має з'явитися сповіщення (alert) з текстом "Ви натиснули на кнопку!". Використайте вбудований JavaScript для реалізації цієї інтерактивності.
